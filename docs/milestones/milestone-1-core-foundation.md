# Milestone 1 — Core Application Foundation

This milestone establishes the minimum viable backend, frontend, and data model needed to run a basic fantasy‑football experience.

---

## Status

- status: `in-progress`
- owner: `product + engineering`
- source of truth for active progress: `docs/milestones/README.md` and linked child issue files under `issues/`

---

## Scope

- Backend scaffolding using Python (FastAPI)
- Frontend scaffolding using a modern JS framework (React)
- Initial database schema for users, leagues, teams, and rosters
- Basic CRUD endpoints
- Local Windows development environment
- GitHub repo structure and branching model

---

## Completion Criteria

- [ ] Application runs end‑to‑end locally
- [ ] Database migrations function correctly
- [ ] UI interacts with backend
- [ ] Codebase avoids Windows‑specific assumptions
